  • OpenClaw 本质是围绕 Coding Agent 套壳的最佳实践
  • 核心机会：谁能把套壳做成普通人也能用的产品

04

Proactive Agent = "管 AI 的 AI"

"Manus 是人管 AI，OpenClaw 是 AI 管 AI"
  • 第一层：可重复执行的定时任务（被动触发的主动）
  • 第二层：基于历史交互主动发现问题（真正的 proactive）
  • 终极形态：能自己提需求、搭 Agent、释放 Agent 的管理者

05

Coding Agent 进入第三阶段

"AI 指挥 AI 写代码"
  • 第一阶段：Copilot（代码补全）
  • 第二阶段：Claude Code 式 Coding Assistant
  • 第三阶段：AI 自己发现需求、澄清需求、调度执行

06

创业方法论：预判为辅，跟随为主

"不要去解决人们还没遇到瓶颈的问题"
  • 预判有效期从半年缩短到一两个月
  • 关键心态：放下 ego，不为差异化而过度坚持
  • 7 人团队产出 ≈ 过去三五十人
07

Agent 时代的商业模式变化

"按 Token 消耗分层，而非组织人数"
  • 日消耗 1 亿 Token ≈ SaaS 时代的世界 500 强
  • 一个人/小组织年贡献 10 万美金成为可能
  • 生产能力越丰饶，有意思的 idea 反而越稀缺
08

新概念：Agent Harness

"越好的马越狂野，越需要马鞍"
  • 类比马鞍——约束和引导 Agent 的脚手架
  • 直接面向终端用户，跳过传统分层
  • 让用户根据自身业务定制 Agent 系统

一句话总结

Coding Agent 的能力已接近天花板，核心竞争从 "做更聪明的 Agent"转向 "让 AI 管理 AI"的产品化和平权化。
创业者需要放下 ego、跟随明确趋势、解决当下真实瓶颈。
