🔥 周度洞察
1 AI编程范式跃迁：从"辅助写代码"到"Agent完成任务"

本周三个关键数据点汇聚——Cursor ARR三月翻番达20亿美元、Claude Code仅8个月登顶、55%工程师常用Agent。与此同时，Vibe Coding在小红书爆发（3794赞6516收藏），Claude Code Review将问题发现率从16%提升至54%、成本低2000倍。

AI编程已从"补全代码"进化到"理解需求+自主实现+审查修复"的完整闭环。

3/9 调查揭示55% Agent使用率，Staff+级达63.5%
3/11 Cursor ARR破20亿美元，三月翻番
3/12 Claude Code Review量化数据出炉
3/14 Vibe Coding小红书爆发3794赞6516收藏
趋势判断
2026年是AI编程从"1.0辅助模式"到"2.0 Agent模式"的转折点。核心技能从"写代码"变成"定义问题+指导Agent+审核结果"。
2 具身智能进入"国资+IPO"双轮驱动期

银河通用25亿、星动纪元10亿、灵初智能20亿——一周内三笔重磅融资，且投资方从市场化基金转向国家级资本（国家大基金三期、中国石化、央视融媒体基金）。六家公司排队IPO，最快5-6月上市。

灵初智能更是00后创业团队，用低成本触觉手套（传统1/10）解决数据采集瓶颈。

趋势判断
具身智能正在复制2015-2016年自动驾驶的融资狂潮但节奏更快。关键竞争维度不是"造更好的机器人"而是"获取更好的数据"。2026年是"上市红利窗口期"。
3 AI军事化路线分化：OpenAI全面拥抱 vs Anthropic坚守红线

本周AI军事化博弈全面升级。Anthropic被国防部列入风险名单后起诉政府、OpenAI和Google员工联名声援（竞争对手罕见站台）。同时OpenAI继拿下五角大楼后向北约32国推销。FDA的Elsa平台因政治因素从Claude切换到Gemini——AI供应商的伦理立场直接影响政府合同。

趋势判断
AI公司与政府的边界博弈正式开始。两条路线的分化将影响整个AI行业的发展方向，企业AI采购需将政治风险纳入决策框架。
