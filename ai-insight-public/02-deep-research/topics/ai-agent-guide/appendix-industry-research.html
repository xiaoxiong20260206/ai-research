调研核心发现

2025年是 AI Agent 商业化应用元年

全球科技巨头和创业公司都在加速布局 AI Agent 赛道。国外以 Microsoft Copilot、Salesforce Agentforce、Anthropic Claude 为代表，国内以 Manus、字节豆包、阿里通义、百度文心为代表。预计到 2026 年，60% 的企业将部署 AI Agent，数字员工正在从概念走向规模化落地。

A.1

数字员工的三大主流模式

模式全景图
模式 核心特征 适用场景 代表产品 自主性
Copilot 模式 人主导，AI 实时建议和辅助 代码编写、文档处理、数据分析 GitHub Copilot、Microsoft 365 Copilot 低
Agent 模式 人监督，AI 自主规划和执行 调研报告、项目部署、复杂任务 Manus、Devin、Claude Code 高
平台模式 企业级 Agent 开发和管理平台 客服、营销、供应链、HR Salesforce Agentforce、BetterYeah 可配置
模式演进趋势

行业正在从 Copilot（辅助）→ Agent（自主执行） 演进。据甲子光年智库预测，到 2026 年认知型 Agent 将覆盖 70% 的企业复杂决策场景。Copilot 是起点，Agent 是终局。

能力：文档、邮件、PPT、Excel 智能处理

Anthropic：Claude + MCP 协议

🔧

MCP（Model Context Protocol）

  • 开源的 AI 工具调用标准协议
  • 已被 OpenAI 等主流厂商采纳
  • 让 Agent 能调用外部工具和 API
  • 成为 Agent 生态的关键基础设施
  • Claude 企业版支持 128K 上下文
🖥️

Computer Use（操作电脑）

  • Agent 可直接操作浏览器和桌面
  • 模拟人类的点击、输入、滚动
  • 支持复杂的跨应用工作流
  • 从"对话 AI"升级为"操作 AI"
  • 开启"数字员工"真正落地

A.3

国内头部玩家与案例

Manus：中国 AI Agent 的"GPT 时刻"

2025 年 3 月，Manus（Monica.im 出品）引发行业关注，被称为中国 AI Agent 的"GPT 时刻"。其核心突破在于工具链整合能力的规模化跃升。

对比维度 Manus 传统 ChatGPT/Claude
核心能力 主动调用浏览器、代码编辑器等工具完成任务 通过文本对话提供建议，无法直接操作
运行环境 云端独立计算环境，无需用户监督 依赖用户提供的本地环境
协作模式 类人类同事：可中途调整，支持偏好记忆 工具型交互：需逐步指导
成果交付 直接生成最终文件（报告、图表、代码包） 提供文本/代码建议，需人工执行

大厂布局：BAT + 字节

竞争维度 OpenAI Google 新锐产品
护城河 强大的模型基座 深耕数据搜索 用户体验 + 数据沉淀

代表 GPT-4o、Operator Gemini、Vertex AI Manus、Cursor、Perplexity

调研总结

  • 三大模式：Copilot（辅助）→ Agent（自主）→ 平台（企业级），正在从左向右演进
  • 海外标杆：Microsoft Copilot、Anthropic Claude+MCP、Salesforce Agentforce、Google Vertex AI
  • 国内标杆：Manus、阿里通义、百度文心、字节豆包，以及垂直领域的 BetterYeah、卓世科技
  • 落地前沿：金融、医疗、工业制造行业渗透率最高，预计 2026 年 60% 企业将部署 Agent
  • 成本拐点：DeepSeek 将推理成本降低 30 倍，推动 AI Agent 从实验走向规模化商用
