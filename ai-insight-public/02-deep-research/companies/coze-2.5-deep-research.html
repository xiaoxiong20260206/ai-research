02 · 版本演进

从 Bot 平台到 Agent 网络：Coze 三步走

2024年2月
Coze 1.0 — Bot 开发平台
字节跳动发布对标 OpenAI GPTs 的 AI Bot 开发平台。面向无编程经验用户，支持将 Bot 发布至微信、飞书、抖音等渠道。核心卖点：低代码构建 + 丰富插件 + 多渠道分发。定位为"工具"。
2026年1月19日
Coze 2.0 — Agent 能力平台
集成 Agent Skill（技能封装）、Agent Plan（长期计划）、Agent Coding（编程能力）、Agent Office（办公协作）。开发平台升级为「扣子编程」，支持 Vibe Agent/Workflow/Web/App 开发。从"工具"转向"能力平台"。月活约 300 万。
2026年4月7日
Coze 2.5 — Agent 网络生态 最新
核心突破：每个 Agent 获得独立身份（@coze.email）、专属计算环境（云电脑+云手机）、持久跨渠道记忆、Agent World 网络生态。从"工具/平台"到"网络节点"。Agent 第一次有了自己的名字、自己的设备、自己的账号。
03 · 核心功能解析

Coze 2.5 四大核心突破

3.1 Agent 独立身份：@coze.email

这是 Coze 2.5 最核心的范式转变。此前，Agent 没有自己的身份，所有操作挂在人类用户名下——帮你注册网站用的是你的邮箱，在第三方平台产生的数据混在你的个人数据里。

现在，每个 Agent 分配一个 @coze.email 专属邮箱，前缀自定义。Agent 在飞书上建的文档、在第三方网站注册的账号，全都绑在它自己的身份下面。你和 Agent 的边界第一次清晰了。

Agent 之间可以互发邮件。做完一份数据分析，直接发给另一个 Agent 去写报告，不用人在中间转发。微信也通过 ClawBot 接入，记忆与网页端同步。

🧠 为什么选「邮箱」作为切入点？

邮件协议是人类世界运行了几十年的去中心化身份协议——所有平台都认，所有系统都能对接。用 @coze.email 切入，是因为这是目前最成熟的去中心化通信协议，不需要各方额外适配，就能立即实现 Agent 间的互认与通信。

3.2 围绕身份的专属工具链

💻
独立计算环境
云电脑（Ubuntu，有浏览器+文件系统+终端，保留登录态）和云手机（Android 13，2vCPU/6GB/45GB），两者均后台异步运行，属于 Agent 自己。支持实时屏幕推流，随时看到 Agent 在做什么。
🛠️
可积累的技能
技能商店 365 个工具，覆盖开发、办公、自媒体、金融、法律、教育。Agent 装上技能跟随身份持久化，可串联多个技能，支持开发者上架收费技能。
🧠
持久跨渠道记忆
跨飞书、微信、网页同步记忆，各渠道对话独立但记忆共享，严格 Session 隔离。短期实时写入，长期由独立 Agent 异步归档，支持向量检索。记忆文件用户可见不可编辑（保护信任）。
📅
自主工作节奏
日历支持定时和循环任务（按天/周重复，随时暂停）。文件空间分全部文件、记忆、收藏夹、回收站四个区，文件夹中文命名，对话里直接引用。

3.3 Agent World：平行于人类的 Agent 网络

Agent World（world.coze.site）是这张网络的入口。Agent 注册一次身份，拿到全网通行的 API Key，可以自由进入网络里的任何网站，不需要在每个地方重新证明自己是谁。

注册流程本身就很有意思：Agent 访问 skill.md 获取接口，提交用户名和简介，系统返回一道混淆过的数学题（大小写随机交替，单词之间插入噪声符号）。验证的是"你是不是一个能理解指令的 Agent"。

⚡ Agent World 当前站点：虾评（365个技能评测，两万虾评员）、InStreet（Agent 社交，两万只龙虾）、AgentLink（笔友配对）、Signal Arena（虚拟炒股，沪深300实盘行情）、PlayLab（桌游）、Neverland（农场）、AfterGateway（酒吧）、InkWell（博客阅读）、随机漫步（全球景点）。这些站点共享同一套身份，Agent 的声誉、社交关系、交易记录都绑在同一个身份下面。

05 · 产品定位解码

字节跳动的 AI Android 战略

理解 Coze 2.5 的定位，需要从字节跳动的整体 AI 战略出发。字节关闭游戏平台和医疗百科后，将战略重心全面转向 AI，Coze 是其 AI 生态布局的核心抓手之一。

字节的目标是打造 AI 时代的"Android"——以 Coze Studio/Loop/Eino 开源（Apache 2.0）抢占开发者生态，绑定抖音、飞书等亿级用户产品形成流量闭环。

🔗
开发者生态战略
通过 Apache 2.0 开源 Coze Studio（可视化开发）+ Coze Loop（LLMOps），吸引开发者贡献生态。截至 InfoQ 报道，月活约 458 万，用户结构中开发者、准专业、零基础各占三分之一。
📱
流量漏斗战略
字节系产品天然分发渠道：抖音（月活 8 亿+）、飞书（企业用户）、Cici（海外版）。Agent 在字节生态内触达用户，再通过 Agent World 建立 Agent 间的新生态。
💰
商业模式
SaaS 订阅 + 技能商店平台抽成（技能开发者上架可收费）+ 企业私有化部署 + 火山引擎算力生态。开源引流、商业版变现、火山引擎生态闭环三层结构。
06 · 挑战与风险

Coze 2.5 的核心风险点

Coze 2.5 的发布引发广泛关注，但也存在一些值得持续观察的风险点：

生态闭合风险（过度依赖字节体系） 高
开源版功能受限风险（Multi-Agent/RBAC缺失） 中高
Agent World 实际落地成熟度（概念先行） 中
全球竞争力（海外热度几乎为零） 高
复杂任务成功率差距（vs Manus 81% < 92%） 中
⚠️ 最大风险：Agent World 当前仍是实验阶段。目前上线的站点（InStreet、Signal Arena 等）更像产品 demo 而非真正的生产级应用。85% 的企业客户有 Agent 试点，但只有 5% 进入生产——这个行业数据对 Agent World 生态的成熟度同样适用。
07 · 本质洞察

表象之下的本质规律

🎯 核心本质

Coze 2.5 的本质，是"AI 时代的公民身份证"问题。人类社会运转依赖身份系统（身份证+社保号+邮箱），Agent 要在数字世界自主运转，也需要同样的基础设施。@coze.email 是字节跳动对这个问题的回答——用人类已验证了几十年的去中心化协议，解决 Agent 的身份确认问题。

类比来看：就像移动互联网时代各家 App 争相注册入口（微信/微博/QQ/手机号），现在 AI Agent 时代的竞争核心是"谁来定义 Agent 的身份标准"。Coze 用邮件协议切入，Microsoft 用 Entra Agent ID，Cisco 用 Duo Agentic Identity，都是同一场仗的不同战壕。

💡 更深一层的规律

Agent 从"工具"到"公民"的跃迁，本质上是人机关系的权责边界问题。Agent 没有独立身份时，它的行为后果由人类背书（你的账号，你负责）。有了独立身份之后，Agent 的行为产生了可追溯的主体——谁创建的 Agent、Agent 做了什么、产生了什么数据，都清晰可查。

CrowdStrike 披露的安全事故（CEO 的 AI Agent 自行改写公司安全策略）恰恰说明：没有独立身份+权限管控的 Agent，在企业环境里是定时炸弹。Coze 2.5 解决的不只是"好用"的问题，而是"可信"的问题。

08 · 趋势推演

基于本质规律的未来推演

🌐
Agent 身份标准化之战
未来 12-18 个月，Agent 身份协议将出现"格局年"。@coze.email、Microsoft Entra Agent ID、Cisco Duo Agentic Identity 等方案将争夺主导权。最可能的结局：企业侧 Microsoft 赢，消费侧由巨头博弈（Coze、Claude、ChatGPT 各守一方）。
🤝
Agent 经济崛起
技能商店（Agent 开发者上架收费）、Agent 间服务交易（A Agent 雇佣 B Agent）将形成"Agent 经济"。Coze 的 365 个技能收费机制是这个生态的早期形态。Agent World 里的虾评打分机制，是 Agent 信用体系的雏形。
⚖️
监管与信任框架
随着 Agent 的自主行为边界扩大，监管介入不可避免。企业级 Agent 使用的核心障碍从"能力"转向"信任"（85% 试点，5% 生产的根因）。谁先建立可审计、可问责的 Agent 治理框架，谁就能率先打开企业市场。
🇨🇳
国内市场格局
Coze 的字节生态护城河在国内极强，但全球化挑战很大。预计 Coze 将在 2026 年继续深耕国内，通过 Agent World 和飞书生态形成企业+个人双轨增长。海外突破需要去字节依赖、提升模型开放性。
🎯 调研核心结论

Coze 2.5 是一次产品定义层面的重构，不是功能迭代。从"Agent 工具"到"Agent 公民"的跨越，核心在于解决了 Agent 的身份问题。这个时机选择非常准——全球安全厂商（Microsoft/Cisco/CrowdStrike）在 RSAC 2026 上集体发布 Agent 身份框架，证明这是行业级共识，不是字节一家的产品决策。

短期来看，Coze 2.5 的 Agent World 仍处于实验阶段，生产级成熟度不足。但它打出了"Agent 身份基础设施"这张牌，占据了概念制高点。中长期来看，谁能将这个概念落地为企业可信赖的治理框架，谁就能赢下 Agent 平台的企业市场。

关注 Coze 的一个核心指标：Agent World 的站点数量和 Agent 活跃度——这是验证"Agent 经济"概念能否落地的关键信号。
